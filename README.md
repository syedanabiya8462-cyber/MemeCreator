# MemeMaster - AI-Powered Meme Creator Platform

A full-stack web application for creating, customizing, and sharing memes using AI technology.

## Features

- 🤖 AI-powered meme generation with DALL-E 3
- 📝 Smart caption suggestions with GPT-4
- 🎨 Advanced meme editor with text overlays
- 📚 50+ classic meme templates
- 🔥 Trending memes feed
- 👤 User profiles and social features
- 📱 Fully responsive design
- 🌙 Dark/Light mode

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **UI Components**: shadcn/ui
- **Backend**: Next.js API Routes
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **AI**: OpenAI API (GPT-4, DALL-E 3)
- **Storage**: Supabase Storage
- **Deployment**: Vercel

## Prerequisites

- Node.js 18+
- npm or yarn
- Supabase account
- OpenAI API key
